# Protocol

## Components


| Name           | CAS-Number | Amount  |
|----------------|-----------|---------|
| Soluble starch | 65996-62-5 | 20 G/L  | 
| K2HPO4         | 7758-11-4 | 1 G/L   |
| KNO3           | 7757-79-1 | 2 G/L   |
| MgSO4 x 7H2O   | 10034-99-8 | 0.5 G/L |
| CaCO3          | 471-34-1  | 3 G/L   |
| NaCl           | 7647-14-5 | 0.5 G/L |
| FeSO4 x 7H2O   | 7782-63-0 | 0.18 G/L |
| MnCl2 x H20    | 64333-01-3| 0.1 G/L |
| ZnSO4 x H2O    | 7446-19-7 | 0.1 G/L |
| Agar powder   | 9002-18-0  | 20 G/L |


## Steps

- Suspend components in demineralized water
- Autoclave at 121 °C for 20 mins

## Notes

- To prepare the liquid version of the medium, simply omit agar
- Starch can form aggregates - mix vigorously
