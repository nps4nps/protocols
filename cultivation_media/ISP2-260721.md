Protocol
=======

### Reagents


| Name          | CAS-Number | Amount  |
|---------------|-----------|---------|
| Yeast extract | 8013-01-2 | 4 G/L   | 
| Malt extract  | 6363-53-7 | 10 G/L  |
| Dextrose      | 77938-63-7 | 4 G/L  |
| Agar powder   | 9002-18-0 | 20 G/L  |


### Steps

- Mix components with demineralized (DEMI) water
- Autoclave using a program with 121 °C for 20 mins.

### Notes

- 1 mL/L [Trace Element Solution](trace-element-solution-260720.md) can be added if necessary
- Compatible with [soil extracts](soil-extract-260721.md)
